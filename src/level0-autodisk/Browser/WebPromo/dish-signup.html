Signing up for DISH Network
Now call 1-800-333-DISH (3474) to sign up for DISH Network programming. You'll be asked which programming packages you want. You'll also need the following:
  • Receiver ID: &sat-nfo-rid;
  • Smart Card: &sat-nfo-cid;
Choose More Info to learn about the different programming packages available.

After you've subscribed, select Continue.
